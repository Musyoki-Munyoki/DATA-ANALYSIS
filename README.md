# 📊 Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn in a telecom company to identify key factors that influence customer retention and provide actionable business insights.

## 🎯 Problem Statement
Customer churn reduces company revenue. The goal of this project is to understand why customers leave and suggest strategies to improve retention.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Key Insights
- Month-to-month contracts have the highest churn
- New customers are more likely to leave early
- Higher monthly charges increase churn risk
- Fiber optic users show higher churn rates
- Electronic check users churn more than auto-payment users

## 📈 Business Recommendations
- Encourage long-term contracts
- Improve early customer onboarding
- Review pricing strategies
- Promote automatic payment methods
